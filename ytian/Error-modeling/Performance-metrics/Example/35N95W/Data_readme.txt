
Data files and program used for Fig. 1

x.dat:		reference data 
y1.data: 	measurements 1 for panels a) and c)	
y2.data: 	measurements 2 for panels b) and d)	

Each data file is a single column text file. 

read_plot_data.R:	R program to read and plot the data and display the metrics in Fig. 1.
